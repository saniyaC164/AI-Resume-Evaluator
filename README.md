#AI Based Resume Evaluator 

Tech Stack: Next.js(Frontend), Flask-Python(Backend)

##Features: 
1.Upload resume in pdf/doc/docx format 
2. Evaluate uploaded resume
3. Show results such as : Strengths, Weaknesses, Suggestions
